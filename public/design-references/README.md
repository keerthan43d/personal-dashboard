# Design references

Drop your best-performing LinkedIn image posts here (the ones whose style you
want the Art Director agent to learn from).

- Formats: `.jpg`, `.jpeg`, `.png`, `.webp`
- Sweet spot: 3–8 images
- Name them anything — order doesn't matter

These are vision-analyzed once into a written "style bible"
(`lib/linkedin/design-style-bible.ts`) that the image Art Director agent draws
on. Add or replace images here and re-run the analysis to refresh the style.
